Lenguajes y Frameworks con los que he Trabajado

CSS3

CSS3 es el lenguaje en casacada de HTML que nos permite decorar nuestras paginas HTML

HTML

“HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto”

Java

Java es un lenguaje de programación y una plataforma informática, es orientado a objetos e imperativo.

JavaScript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

php

PHP es un lenguaje de programación de uso general que se adapta especialmente al desarrollo web.

Python

Python es un lenguaje de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código.​ Se trata de un lenguaje de programación multiparadigma, ya que soporta parcialmente la orientación a objetos, programación imperativa y, en menor medida, programación funcional.

C/C++

C es un lenguaje de programación de propósito general, C++ es un lenguaje de programación.La intención de su creación fue extender al lenguaje de programación C mecanismos que permiten la manipulación de objetos. En ese sentido, desde el punto de vista de los lenguajes orientados a objetos, C++ es un lenguaje híbrido.​
